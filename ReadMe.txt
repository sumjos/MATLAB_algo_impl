This contains code algorithm implementations of different numerical techniques in Math

1. RRT* algorithm with obstacle avoidance for a track

2. Iterative methods of solving system of equations: Jacobi, Gauss-Seidel, SOR

3. Numerical integration methods like: Euler Method, R-K 2nd order and AB-4th order taking y' = 2y as an example and generates a table with integration data

4. Example of GaussElimination with partial pivoting

5. Solution of Tridaigonal system equation matrix
